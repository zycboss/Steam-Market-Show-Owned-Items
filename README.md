# Steam-Market-Show-Owned-Items
Automatically checks your Steam inventory and shows owned item counts (red [xN]) while browsing the Community Market.

Only works on the legacy market page, not working in the new BETA version.

Tested games：
- Steam inventory
- Dota2
- CS2
- TF2
- Battle Block Theater
- Banana

Unsupported game：
- Don't starve together

Please issue feedback if any other game is not working.
